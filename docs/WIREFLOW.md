## UI Flow Summary

### Primary navigation
- **Setup** (host)
- **Collect Codes** (host)
- **Inspiration** (question ideas)

### Setup (host)
1. Enter game title.
2. Add players and questions.
3. Choose scoring mode (simple or weighted).
4. Lock the game to generate the player share link.

### Player flow
1. Select a player name.
2. Rank players for each question.
3. Copy the generated secret code.

### Collect codes (host)
1. Paste each `name:code` line.
2. Confirm submission status.
3. Generate the Game Night link and QR.

### Reveal
- Fullscreen prompt → question → ranking reveal → round score → total score → finale.
