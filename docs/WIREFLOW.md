## UI Flow Summary

### Primary navigation
- **Setup** (host)
- **Collect Codes** (host)
- **Inspiration** (question ideas)

### Setup (host)
1. Enter game title.
2. Add players and questions.
3. Choose scoring mode.
4. Lock the game to generate the player link.

### Player flow
1. Select a player name.
2. Rank players for each question.
3. Copy the generated player code.

### Collect codes (host)
1. Add each player code one at a time.
2. Confirm submission status.
3. Remove no-shows if needed.
4. Generate the reveal link and QR.

### Reveal
- Fullscreen prompt
- Question
- Ranking reveal
- Occasional subtotal
- Finale intro
- Final standings
