Ranking Your Friends

A playful party game: pick juicy questions, rank each other, and reveal the funniest results round by round.

No accounts, no logins. The host locks the game, players send a short secret code, and the host runs a big-screen style reveal.

How it works

  1. Create the player link Add names and questions, then create one link for everyone.
  2. Collect player codes Each friend ranks the group privately and sends one player code back.
  3. Open the reveal Generate the reveal link and run the big-screen presentation.
